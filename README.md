# Automated Inventory Auditor

## Overview
The Automated Inventory Auditor is a Python-based automation tool that reads inventory data from a CSV file, analyzes stock levels and expiration status, and generates a structured audit report.

This tool is designed to help small businesses, pharmacies, and retail stores quickly identify:

- Expired products
- Out-of-stock items
- Items that require restocking
- Disposal requirements

## Features
- Reads structured inventory data from CSV
- Detects expired items
- Detects zero or negative stock levels
- Automatically generates a professional audit report (audit_report.txt)
- Clean and modular code using functions

## Technologies Used
- Python
- CSV module
- File handling
